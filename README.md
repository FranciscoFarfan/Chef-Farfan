# 👨‍🍳 Chef-Farfan Telegram Bot

Un bot de Telegram interactivo diseñado como proyecto de demostración del uso de bots de Telegram. Funciona como un asistente culinario que permite buscar recetas de cocina paso a paso, o bien, descubrir qué platillos se pueden preparar utilizando únicamente los ingredientes disponibles en casa.

## ✨ Características Principales
- **Búsqueda por Nombre:** Obtiene detalles específicos de una receta (tiempo de preparación, porciones y lista exacta de ingredientes).
- **Asistente de Ingredientes:** En base a una lista de ingredientes ingresados, el bot sugiere posibles recetas que los incluyan.
- **Consultas a Base de Datos:** Integra una base de datos SQL para almacenar y recuperar de forma eficiente múltiples recetas y sus respectivos ingredientes y cantidades.

## 🛠️ Tecnologías y Herramientas Utilizadas
- **Backend:** PHP 7+
- **Base de Datos:** MySQL / MariaDB
- **Integraciones:** Telegram Bot API (vía Webhooks empleando la librería `telegram-bot-sdk`)
